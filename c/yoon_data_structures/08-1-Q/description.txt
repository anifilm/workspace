문제 08-1 [이진 트리의 소멸]
우리가 구현한 이진 트리에는 소멸관련 함수가 정의되어 있지 않다. 때문에 예제에서는 동적으로 할당된 노드를 소멸시키지도
않고 프로그램을 종료하얐다. 따라서 이진 트리를 완전히 소멸시키는 함수를 다음과 같이 선언하고 정의하고자 한다.

  void DeleteTree(BTreeNode* bt);

위 함수가 정의되면, 이는 다음과 같이 호출되어야 한다.

  int main() {

    BTreeNode* bt1 = MakeBTreeNode();

    ......

    DeleteTree(bt1);

  }

그리고 위와 같이 DeleteTree 함수가 호출되면, bt1이 가리키는 노드를 루트 노트로 하는 트리 전부가 완전히 소멸되어야
한다. 그럼 앞서 완성한 BinaryTree2.h와 BinaryTree2.c에 위 함수를 선안하고 정의해 보자.
